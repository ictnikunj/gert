Mit diesem Plugin kannst du in den Erlebniswelten Kategorien oder Unterkategorien inklusive Bild auflisten und verlinken. Auf der einen Seite kannst du so auf Kategorieseiten jeweils automatisch alle Unterkategorien anzeigen lassen. Dadurch ist die Navigation in deinem Shop für deine Besucher insbesondere bei mehreren Kategorie-Hierarchien übersichtlich und einfach. Auf der anderen Seite kannst du auch beliebige Kategorien verlinken. So können zum Beispiel direkt auf der Startseite deine wichtigsten Kategorien vorgestellt werden.

Die Darstellung kann direkt über Eingabe der CSS-Klassen gesteuert werden. Durch das Bootstrap Framework können so beliebige Darstellungsformen für alle Bildschirmgrössen erreicht werden. Für Entwickler ist es zudem sehr einfach möglich, die Templates in einem individuellen Theme zu überschreiben und so die Darstellung beliebig zu konfigurieren.
